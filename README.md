# Eventos Chiapas

## Descripción del proyecto
Eventos Chiapas es un sitio web estático desarrollado para una agencia local dedicada a promocionar eventos culturales, artísticos y recreativos del estado de Chiapas.

El sitio contiene:
- Página principal: `index.html`
- Página de eventos: `eventos.html`
- Página de contacto: `contacto.html`

## Objetivo general
Aplicar el uso de Git y GitHub en un entorno colaborativo realista, usando ramas, commits, pull requests, resolución de conflictos, documentación y publicación con GitHub Pages.

## Instrucciones de uso
1. Clonar el repositorio:
   ```bash
   git clone URL_DEL_REPOSITORIO
   ```
2. Entrar a la carpeta:
   ```bash
   cd eventos-chiapas
   ```
3. Abrir el archivo `index.html` en el navegador.

## Publicación en GitHub Pages
El sitio se publicará desde la rama `main` usando GitHub Pages.
